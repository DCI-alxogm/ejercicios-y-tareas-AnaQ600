Para poder actualizar el repositorio primero hacemos un archivo o si ya lo tenemos creado los editamos, después escribimos el "git add" y escribir el nombre del archivo y dar enter, despues escrbir el "git commit" y escribir un pequeño tento en la parte de arriba, luego dar Control O, enter y despues Control X. Despues escribir git push, escribiur nuestro usuario y contraseña y listo
